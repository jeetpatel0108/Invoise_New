## HOW TO USE

### Creating Invoices:
1. Select Payment Mode (CASH or ONLINE)
2. If ONLINE, enter payment reference (optional)
3. Add items and generate invoice

### CASH Invoice:
   • No QR code in PDF
   • Shows payment mode and status
   • Shows authorized signatory

### ONLINE Invoice:
   • QR code in PDF footer
   • Shows payment reference
   • Shows payment mode and status

### Managing Payments:
1. Go to "Payment Tracking" on dashboard
2. View payment statistics
3. Search unpaid invoices (filter by status, customer, date, bill #)
4. Click "Mark as Paid" to update status
